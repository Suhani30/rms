1. git init
   GIT initialize

2. git status
   check GIT status for each commit

3. git add --all
   add files to local git repository

4. git log
   checks for any commits made in the repository

5. git config --global user.email "you@email.com"
   git config --global user.name "Your Name"
   used to create a user name and a email address for validation to work between the    local and remote repository.

6. git commit -m "message"
   write a comment like message for each commit to understand the flow of the    project.

7. git remote add origin <project URL>
   add your local repo with your remote repo

8. git push -u origin master
   add any new files or updated files to your github remote repo

9. git pull --rebase origin master
   use this command if you see any error after using git push -u origin master

10. exit
    exits from git bash
